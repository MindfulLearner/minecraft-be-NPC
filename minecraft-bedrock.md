# Minecraft: Bedrock Edition

## Overview

Minecraft: Bedrock Edition is the cross-platform version of Minecraft built on the **Bedrock codebase** — written in C++ to run on platforms that cannot support Java. It is published by Xbox Game Studios and developed by Mojang Studios, and is the most widely available version of Minecraft across modern devices.

## Supported Platforms

Bedrock Edition runs on more platforms than any other edition:

- **PC:** Windows 10 / 11
- **Console:** Xbox One, Xbox Series S|X, PlayStation 4, PlayStation 5, Nintendo Switch
- **Mobile:** Android, iOS
- **Other:** Amazon Fire, ChromeOS

> Java Edition is limited to Windows, macOS, and Linux only.

---

## Key Features

### Cross-Platform Multiplayer
All Bedrock players — regardless of device — can play together online. A Nintendo Switch player can join a session with a PlayStation or mobile player seamlessly. Java and Bedrock players **cannot** cross-play with each other.

### Performance
Because it is written in C++, Bedrock runs more efficiently than Java Edition, especially on lower-end hardware and mobile devices. It handles larger render distances and more players with less lag on modest specs.

### Controller Support
Bedrock natively supports game controllers on all platforms. Java Edition is keyboard-and-mouse only.

### Marketplace
The **Minecraft Marketplace** is an in-game store where creators sell and share content using **Minecoins** (the in-game currency). Content includes:

- Skin packs
- Texture/resource packs
- Worlds and maps
- Add-ons (new mobs, blocks, items, recipes)

A **Marketplace Pass** subscription gives access to a rotating library of premium content for a monthly fee.

### Add-Ons
Add-ons are Bedrock's equivalent of mods. They can introduce new blocks, entities, items, and game mechanics. They sync to your Microsoft Account and work across all your devices and platforms. In multiplayer, only the world host needs to have the add-on installed — other players benefit automatically.

### Realms
Minecraft Realms is an official hosted server service. Bedrock Realms support cross-platform play and include an automatic backup system.

### Splitscreen (Console)
Console versions of Bedrock support local splitscreen multiplayer — a feature not available in Java Edition.

## Bedrock vs. Java Edition — Quick Comparison

| Feature | Bedrock | Java |
|---|---|---|
| Platforms | Windows, Xbox, PS4/5, Switch, Mobile | Windows, macOS, Linux |
| Cross-play | Yes (all Bedrock devices) | No |
| Language | C++ | Java |
| Controller support | Yes | No |
| Mods | Add-ons + Marketplace | Full mod ecosystem (Forge, Fabric, etc.) |
| World format | LevelDB | Anvil |
| Local splitscreen | Yes (console) | No |
| Dev snapshots | Preview builds | Snapshots |
| Performance | Better on low-end hardware | Higher ceiling on high-end PC |

## Technical Notes

- ~90% of the Bedrock codebase is shared across all platforms; only platform-specific layers (networking, storage) differ
- World data is stored in **LevelDB** format
- Microsoft Account required for cross-play and Marketplace purchases
- Add-ons sync automatically via Microsoft Account across devices
